# Social-Media-UI-UX-project-Group-
Project Overview
“The Voice of Sri Lanka” is a mobile-first social media platform designed to connect Sri Lankan users through secure, identity-verified profiles. The application emphasizes user authentication, profile creation, social networking, and community interaction in a localized and intuitive interface.

Key UI/UX Features
1. Secure Onboarding & Identity Verification
Multi-step registration with NIC (National Identity Card) and mobile number verification.
Biometric login option for enhanced security and convenience.
OTP-based authentication via SMS (HelaLink integration).
Document upload flow for NIC front/back and selfie verification, ensuring user authenticity.
2. User-Centric Profile Management
Profile editing interface with fields for bio, education, location, gender, relationship status, and contact details.
Profile verification badge upon successful NIC verification.
“Add to story” feature for ephemeral content sharing.
3. Social Networking & Engagement
Friend system with suggestions, friend requests, and mutual friend counts.
Newsfeed layout with posts, stories, and user activity timelines.
Real-time chat interface with active status indicators and message history.
4. Intuitive Navigation & Menu Structure
Bottom/Top navigation bar with quick access to:
Home (Feed)
Friends
Notifications
Chats
Profile
Menu (Settings, Help, Logout)
Clean and localized UI with Sinhala/English support inferred from context.
5. Notifications & Alerts
Segmented notification panel (New vs. Earlier).
Push notifications for OTPs, friend requests, and messages.
6. Security & Account Management
Password reset flow with OTP verification.
Security updates section for app safety information.
Logout and session management.
Design Principles Applied
Minimalist and clutter-free layout with clear visual hierarchy.
Consistent use of icons, buttons, and input fields for familiarity.
Mobile-responsive design optimized for touch interaction.
Localized content (e.g., use of NIC, HelaLink OTP service).
Accessible text and contrast for readability across age groups.
Target Audience
Sri Lankan residents seeking a secure and verified social platform.
Users interested in local communities, networking, and content sharing.
Individuals preferring identity-authenticated interactions over anonymous social media.
Technology & Platform
Mobile-native design (iOS/Android).
Backend integration for SMS OTP, biometric auth, and document verification.
Cloud storage for profile images and uploaded documents.
Real-time messaging and notification systems.
Goals of the UI/UX Design
Provide a seamless, secure, and engaging user experience.
Foster trust through identity verification.
Encourage social interaction and community building.
Ensure ease of use for non-technical users with intuitive flows.
